# Scraplan 서비스 소개

### **📍 Scrap  + Plan**

```
주말에 무엇을 할 지 고민하고 계시나요?
다른 사람들은 어떻게 하루를 보내는지, 지금 바로 확인해보세요!
```

**Scraplan**은 지도를 기반으로 **해당 장소의 다양한 즐길거리**를 큐레이션 형식으로 제공하고, <br />
해당 큐레이션을 이용해 **나만의 일정**을 편리하게 관리할 수 있는 서비스입니다. <br />
